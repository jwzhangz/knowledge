
###### 泛型术语
```
参数化的类型       List<String>
实际类型参数       String
形式类型参数       E
泛型              List<E>
无限制通配符类型   List<?>
原生态类型         List
有限制的类型参数   <E extends Number>
递归类型参数       <T extends Comparable<T>>
有限制通配符类型    List<? extends Number>
泛型方法           static<E>List<E>asList(E[] a)
类型令牌           String.class
```
